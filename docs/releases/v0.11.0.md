# Forward for Dynatrace v0.11.0

This enterprise preview establishes the clean product and release baseline for controlled evaluation and
non-production use.

## Highlights

- Converts current Dynatrace service dependencies into governed Forward path and intent-check evidence.
- Provides Read Only, Network Operator, and Network Admin workflows with explicit least-privilege behavior.
- Supports deterministic plan, exact-approved create/update, collision protection, mutation budgets, and post-write
  reconciliation.
- Correlates modeled Forward reachability with Dynatrace Site Reliability Guardian outcomes while preserving both
  source-of-truth boundaries.
- Adds tenant-owned dependency discovery profiles, current-evidence validation, bounded results, and fail-closed
  mapping.
- Delivers one tenant-validated Dynatrace app archive with a CycloneDX SBOM, SHA-256 checksums, optional detached
  signature, and GitHub attestations.
- Includes enterprise installation, security, RBAC, operations, acceptance, and release documentation.

## Installable

`forward-dynatrace-app-v0.11.0.zip` is the only installable artifact. The SBOM, checksum, signature, and attestation
records are verification evidence. Forward requires no software from this project.

## Verification

Download all release assets together, run `sha256sum -c SHA256SUMS`, and verify the app and SBOM attestations against
`forwardnetworks/forward-dynatrace` before installation.
